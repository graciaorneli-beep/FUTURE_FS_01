## 📌 Description

Portfolio personnel et professionnel présentant mes compétences, mes projets et mon parcours en tant que développeuse Full Stack Junior. Ce site a été conçu pour être moderne, responsive et accessible sur tous les appareils.

Réalisé dans le cadre du programme **Future Interns – Full Stack Web Development (Task 1, 2026)**.

---

## 🛠️ Technologies utilisées

| Technologie | Usage |
|-------------|-------|
| HTML5 | Structure et contenu |
| CSS3 | Mise en page, animations, responsive |
| JavaScript | Interactivité, animations au scroll |
| [ScrollReveal.js](https://scrollrevealjs.org/) | Animations d'apparition au scroll |
| [Typed.js](https://github.com/mattboldt/typed.js/) | Effet de frappe animé |
| [Boxicons](https://boxicons.com/) | Icônes |

## 🚀 Lancer le projet en local

1. **Cloner le repository**
```bash
git clone https://github.com/graciaorneli-beep/FUTURE_FS_01.git
```

2. **Ouvrir le dossier**
```bash
cd FUTURE_FS_01
```

3. **Ouvrir dans le navigateur**
```bash
# Double-cliquer sur index.html
# Ou utiliser l'extension Live Server sur VS Code
```

Aucune installation ni dépendance requise — le projet fonctionne directement dans le navigateur.
